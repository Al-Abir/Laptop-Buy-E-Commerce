# 💻 Redmi Laptop E-commerce Website

Ei project ekta full-featured online e-commerce website jekhane user ra Redmi laptop browse, order, and checkout korte pare.

---

## 🚀 Live Demo
[Click here to view](https://your-live-site-link.com)

---

## 🛠️ Technologies Used

- **Frontend:** React.js, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
- **Payment Gateway:** (e.g. Stripe or SSLCommerz)

---

## 🔥 Major Features

### 🧑‍💼 User Features:
- ✅ Sign Up & Sign In (JWT based)
- 🛍️ Browse all Redmi laptops
- 🛒 Add to Cart
- ❤️ Wishlist system
- 💳 Secure Checkout & Payment
- 📦 Order Tracking
- 🌐 Responsive Design (Mobile/Desktop)

### 🛠️ Admin Features:
- 📊 Dashboard Overview (Sales, Orders, Users)
- ➕ Add / Edit / Delete Laptop Products
- 🔐 Secure Admin Access
- 📦 Manage Orders
- 💰 Manage Discounts / Offers
